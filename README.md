# Retagram (레타그램)

**인스타그램 스토리를 흔적 없이 확인하세요.**


## 🚀 주요 기능
- **흔적 없는 스토리 시청:** 스토리를 시청해도 상대방의 '읽은 목록'에 내 계정이 표시되지 않습니다.
- **경량 웹뷰어:** 인스타그램 웹 버전을 그대로 사용하여 로그인이 간편하고 안전합니다.
- **가벼운 용량:** 불필요한 기능 없이 스토리 차단 기능에 집중했습니다.

## ⚠️ 주의사항 (필독)
- **개인정보 보호:** 본 앱은 웹뷰를 사용하며, 사용자의 인스타그램 아이디와 비밀번호를 서버로 전송하거나 저장하지 않습니다. 모든 데이터는 인스타그램 서버와 직접 통신합니다.
- **계정 정책:** 인스타그램의 정책 변경에 따라 기능이 차단될 수 있습니다. 

## 📥 다운로드 및 설치
1. [Release 페이지]([https://github.com/사용자계정/레포지토리명/releases](https://github.com/qyronx/Retagram/releases/tag/Retagram))로 이동합니다.
2. `Retagram.apk` 파일을 다운로드합니다.
3. 다운로드한 파일을 스마트폰으로 옮긴 뒤 설치합니다. (설정에서 '출처를 알 수 없는 앱 설치' 허용 필요)

## 🛠 기술 스택
- **Language:** Kotlin
- **Framework:** Android WebView, AppCompat
- **Core Logic:** JavaScript Injection (Fetch & XHR Hooking)

## 💡 사용 방법
1. 앱을 실행하고 본인의 인스타그램 계정으로 로그인합니다.
2. 일반적인 인스타그램 앱처럼 스토리를 시청합니다.
3. 앱이 자동으로 조회 요청을 차단하여 상대방에게 흔적을 남기지 않습니다.
